# ArtiCol
# Требования и рекомендации
1. Требования ко внешнему носителю: необходима файловая система типа ext4, иначе запуск программы непосредствено с внешнего носителя невозможен;
2. Рекомедуется запускать программу через `sudo`.

# Быстрый старт
1. Для полной статической компиляции использовать `go build -tags netgo -ldflags '-extldflags "-static"'`;
2. Если уже есть бинарный файл и Вы находитесь в той же директории, что и файл, то просто запустите его через `sudo /path/to/ArtiCol`

# Стек собираемых данных
1. `/etc/passwd` и `/etc/shadow`;
2. Все логи из `/var/log/`;
3. `.bash_history` каждого пользователя;
4. Все сессии из `/var/run/systemd/sessions`;
5. Все сетевые соединения и все процессы;
6. Все модули из `/proc/modules`;
7. Информация об `OS`, `Kernel`, `Uptime`, `Hostname`.
