# ArtiCol
Требования и рекомендации:
1. Требования к внешнему носителю: формат должен быть "ext4".
2. Рекомедуется запускать программу через `sudo`


Быстрый старт:
1. Для полной статической компиляции использовать `go build -tags netgo -ldflags '-extldflags "-static"'`, где:
- `go build` - начало компиляции проекта.
- `-tags netgo` - флаг, который указывает компилятору использовать чистую Go-реализацию сетевого стека.
- `-ldflags` - флаг для передачи флагов линковщику.
- `'-extldflags "-static"'` - флаг, который указывает линковщику вшить все зависимости в бинарный файл.
2. Если уже есть бинарный файл и Вы находитесь в той же директории, что и файл, то просто запустите его через `sudo ./file_name`
